Virtual Cookbook App

Since March 2020, I've been cooking at home much more often. Seeing that I've accumulated personal recipes over the years, I figured I would build an app so I could access them all on my phone. This IOS application was built with the iPhone 11 Pro in mind, but works with the iPhone 11 Pro Max and smaller screen sizes. For this project I followed an IOS app-building tutorial to learn how to use Swift and build some basic features, then used those skills to create this. Language used: Swift.

Note: The main portion of the development is complete, at this point I am simply continuing to add new recipes and images as well as adjusting some UI features.
