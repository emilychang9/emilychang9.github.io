Alliance Management Tool

After weeks of wanting to build a website, I decided to build this for players of the mobile game "Marvel Contest of Champions." Players can create or join teams and complete game content together, which requires communication and strategy. I created this website to streamline team strategizing and offer a user-friendly environment for players of all ages and experiences. Since the game is generally played on smaller devices (phones, tablets), the screenshots below are on a mobile browser screen rather than a desktop computer. Language used: HTML, CSS, Javascript, PHP.

Note: this project is not complete and I do not own any of the game-related images displayed. I started this project for fun, and believe I am working under the fair use clause. If there are any questions or concerns regarding the content of this project, please contact me.
