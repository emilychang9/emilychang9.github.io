Safe Eats

This web application was built by me and a team of 4 other unviersity students in August 2020 during the virtual hackathon Hack'20. Safe Eats is an app that allows users to search for nearby restaurants and see the local county's COVID-19 statistics. I personally worked on the main website for the application, and helped code the initial version of the API integration. Languages used: React, HTML, CSS.

Note: since we were only given enough funds to use the Google Places API for the 24-hour event, the web application is no longer functional. Our team submission on DevPost is available here.
